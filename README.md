# Serbian Football Portal

Static starter page for GitHub and Cloudflare Pages.

## Files
- `index.html`
- `style.css`
- `app.js`

## Deploy to GitHub + Cloudflare Pages
1. Upload all files to a new GitHub repository.
2. In Cloudflare Pages, create a new project.
3. Connect your GitHub repository.
4. Framework preset: `None`.
5. Build command: leave empty.
6. Build output directory: `/`
7. Deploy.

## Notes
This is a polished static front page. You can later connect real data feeds for:
- live scores
- fixtures
- standings
- latest Serbian football news
